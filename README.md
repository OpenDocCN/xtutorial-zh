## 下载

### Docker

```
docker pull apachecn0/xtutorial-zh
docker run -tid -p <port>:80 apachecn0/xtutorial-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g xtutorial-zh
xtutorial-zh <port>
# 访问 http://localhost:{port} 查看文档
```
